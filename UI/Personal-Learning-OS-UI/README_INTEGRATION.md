# Personal Learning OS — UI prototype

Готовый интерактивный прототип интерфейса Personal Learning OS в направлении «Тихий фокус».

## Что включено

- экран «Сегодня»;
- светлая и тёмная темы с сохранением выбора в `localStorage`;
- учебные пространства и маршрут;
- интерактивная карта знаний;
- библиотека материалов;
- адаптивный календарь;
- AI-наставник;
- сессия повторения;
- адаптивная мобильная версия.

## Основные файлы интерфейса

- `app/page.tsx` — компоненты страниц, навигация и интерактивные состояния;
- `app/globals.css` — дизайн-система, обе темы, компоненты и responsive-правила;
- `app/layout.tsx` — корневой layout;
- `package.json` и `package-lock.json` — зависимости и команды проекта.

## Локальный запуск

Требуется Node.js 22.13 или новее.

```bash
npm install
npm run dev
```

## Перенос в существующий Next.js-проект

1. Установите иконки:

   ```bash
   npm install lucide-react
   ```

2. Перенесите `app/page.tsx` и `app/globals.css` в соответствующую папку приложения.
3. Если в проекте уже есть собственный `layout.tsx`, сохраняйте его и подключите `globals.css` оттуда.
4. Текущая версия использует демонстрационные данные и клиентское состояние. При подключении backend замените массивы и обработчики в `app/page.tsx` на API-вызовы.

## Темизация

Тема переключается атрибутом `data-theme` на корневом элементе `<html>`. Основные дизайн-токены находятся в начале `app/globals.css` в блоках:

```css
:root { /* light theme */ }
:root[data-theme="dark"] { /* dark theme */ }
```

Выбор хранится под ключом `plos-theme` в `localStorage`.

## Важно

Папки `node_modules`, `dist`, служебные файлы хостинга и история Git в архив не включены. После распаковки код можно добавить в существующий репозиторий обычным коммитом.
